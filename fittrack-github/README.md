# FitTrack Pro (упрощённая версия для GitHub Pages)

- Один файл: `index.html`
- Никаких сборок, TypeScript и Vite.
- Локальное сохранение данных через `localStorage`.
- Работает как обычная статическая страница.

## Как развернуть на GitHub Pages

1. Создай новый репозиторий на GitHub.
2. Залей содержимое архива (файл `index.html`) в ветку `main`.
3. В настройках репозитория: **Settings → Pages → Source: `main` / `/ (root)`**.
4. Сохрани настройки — через минуту приложение будет доступно по адресу:

`https://<твой-username>.github.io/<имя-репозитория>/`
